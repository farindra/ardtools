# ArdTools

ArdTools adalah alat penunjang untuk melakukan aktifitas protokol pada jaringan yang berbasis web, alat ini dibuat untuk mendukung kinerja para WebHoster, Developers, Server Administrator dan siapa saja yang membutuhkan alat penunjang ini secara GRATIS.   

### Version
1.0

### Minimum Requirement
* [PHP 5.3]

And of course Dillinger itself is open source with a [public repository][dill]
 on GitHub.

### Installation

Download semua file dan letakan pada web folder yang anda buat, edit file "config.php"

```sh
$ npm i -g gulp
```

```sh
$ git clone [git-repo-url] dillinger
$ cd dillinger
$ npm i -d
$ gulp build --prod
$ NODE_ENV=production node app
```

Beberapa tutorial yang berguna untuk anda:

* [plugins/dropbox/README.md] [PlDb]
* [plugins/github/README.md] [PlGh]
* [plugins/googledrive/README.md] [PlGd]
* [plugins/onedrive/README.md] [PlOd]
